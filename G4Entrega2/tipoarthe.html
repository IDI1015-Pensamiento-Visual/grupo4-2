Collage

Usando recortes de periódicos o revistas, papeles de colores y materiales textiles, se crea un cuadro con imágenes existentes para expresarse.

Medio Gráfico

Utilizando materiales como lápices, tizas, carboncillos, bolígrafos y gomas, se realizan variadas actividades de dibujo con mayor detalle.

Pintura

Ya sea utilizando tinta negra o pintura de colores con mezcladores, el enfoque principal es el uso de del pincel sobre un canvas.

Escultura

Usando de base barro, plastilina, piedra u otro material, se puede crear una obra que se puede ver desde todos los ángulos y que posee un gran potencial de creatividad, se pueden incluir otros materiales como cuerdas, cartones y reciclados para mayor expresionismo y puede ser grupal.
